El editor de Superglus es actualmente compilable con Delphi 7, aunque en principio no debe haber problemas para compilar con versiones anteriores a partir de la 5, o posteriores.

Utiliza los componentes GPL SynEdit (http://synedit.sourceforge.net/), asi como un componente SynAnySyn, tambien GPL y (c) Jan Verhoeven.

El principal cambio en esta version del editor es la substitucion del componente freeware TADVMemo por los indicados, de manera que el codigo completo pueda usar licencia GPL.

No obstante, los planes del autor pasan por convertir en la medida de lo necesario el presenet codigo para que sea compilable con Lazarus (http://lazarus.freepascal.org) de manera que pueda ser compilador para otras plataformas como linux, SOlaris o MacOS X.
